# Proyecto Digital — Compliance y Gobernanza Continua de IA

SaaS B2B de **compliance y gobernanza continua de IA** para el **mid-market**:
inventario de sistemas de IA, clasificación de riesgo (EU AI Act), evidencia lista
para auditoría y monitoreo de cambios regulatorios.

> Nombre comercial por decidir.

## Stack

Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · Vercel (objetivo)

## Empezar

```bash
npm install
npm run dev      # http://localhost:3000
```

- `/` — landing de lanzamiento.
- `/dashboard` — esqueleto del producto (resumen, inventario, riesgo, gap) con datos de ejemplo.
